## About

막상 나와서 뭘먹어야 할지 모르겠거나 내 주변에 있는 맛집만 보고 싶은 경우가 있지 않나요 ?  
맛집 정보를 쉽게 받고 싶을때 음식을 맛있게 먹고 이 집을 공유하고 싶을때  
이런 것들을 간편하고 빠르게 정보를 보여주는 Tell me Restaurant ! 


Tell Me Restaurant 에서는 이런것들을 할 수 있습니다.

* 간편한 로그인
* 태그 기능으로 원하는 지역 맛집 보기
* 마이페이지 에서 내가 올린 글 보기
* 내 주변 맛집을 쉽게 올리기
